## Install

```bash
git clone https://github.com/GAKiknadze/limit_guard.git
cd limit_guard
```

## Run

### Local

#### 1. Install requirements

```bash
pip install requirements.txt
```

#### 2. Run file
```bash
python main.py
```

### Docker

```bash
docker compose up -d
```
